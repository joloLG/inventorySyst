⚡ Quick Configuration Setup

📋 Follow these steps to get your inventory system running:

Step 1: Create Supabase Project

  1. Go to supabase.com
  2. Sign up and create a new project
  3. Wait for the project to be ready (2-3 minutes)

Step 2: Get Your Credentials

  1. In your Supabase project, go to Settings → API
  2. Copy the Project URL
  3. Copy the anon public key

Step 3: Configure Your System

Enter your Supabase credentials below:

Example: https://abcdefgh.supabase.co
Long string starting with eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

Step 4: Set Up Database

After configuration, you'll need to create the database tables:

  1. Go to your Supabase project dashboard
  2. Click on SQL Editor
  3. Copy and run the SQL script from README.md
  4. Wait for all tables to be created
